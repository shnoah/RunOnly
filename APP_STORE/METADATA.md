## Description Draft

RunOnly는 Apple 건강의 러닝 기록을 더 보기 쉽고, 더 러너 친화적으로 정리하는 iPhone 앱입니다.

앱 안에서 바로 확인할 수 있는 내용:

- 월간/연간 러닝 거리 요약
- 러닝 상세 차트와 스플릿
- 심박, 케이던스, 상승 고도 요약
- 러닝 경로와 퍼포먼스 흐름
- 개인 기록(PR) 추적
- 신발별 누적 거리와 착용 기록
- 공유용 러닝 이미지 생성

RunOnly는 계정 생성 없이 동작하며, 현재는 서버 업로드나 외부 분석 SDK 없이 iPhone 안에서 러닝을 분석합니다.
